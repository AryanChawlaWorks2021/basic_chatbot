# 🤖 AI Chatbot

A basic AI-powered chatbot built for **learning and experimentation purposes**.  
This project was created to explore **real-time communication using Socket.IO** and integrate **Google Gemini AI** for generating responses.

The chatbot allows users to send messages through a simple chat interface and receive **AI-generated responses in real time**.

---

## ✨ Features

- 💬 Real-time messaging using **Socket.IO**
- 🧠 AI-generated responses using **Google Gemini API**
- ⚛️ Simple and responsive **React chat interface**
- 🚀 **Express.js backend** for handling server logic
- 🔌 WebSocket-based communication between frontend and backend

## 🎯 Purpose of the Project

This project was built to practice:
